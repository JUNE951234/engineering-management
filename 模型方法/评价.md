# 评价概念
评价是指评价主体根据一定的评价标准，对评价客体的价值高低（有无价值及价值大小）做出判断的活动，也可指某人对某事的认同程度（如对事件、某人、某事物的评价）。

从决策角度看，评价是主观、客观相结合的过程：在评价目标指导下，对被评对象进行系统分析，确定评价指标体系及权重；对被评对象的有关属性进行测定并转化为评价者的主观效用，是决策者进行决策的基础。同时，评价是决策过程中的重要活动，可通过修正偏离价值标准的行为，使其按预定目标和方向发展，管理在一定意义上就是在评价中不断修正与完善的过程。

# 评价的功能
| 功能   | 说明                                                                 |
|--------|----------------------------------------------------------------------|
| 判断功能 | 对评价对象的归属进行区分，判定对象某种属性对特定评价目标的从属程度，表现为评定、认定、评语等形式 |
| 选择功能 | 通过对比事物，形成对优劣程度的认识，选择价值最大的对象作为最优选择对象 |
| 导向功能 | 指出评价对象的不足之处，引导事物向有利于目标实现的方向发展，促使评价对象不断完善 |
| 调控功能 | 对事物发展过程进行跟踪认识，分析事物发展过程偏离目标的程度，进而对事物进行调控 |
| 激励功能 | 通过定性与定量的评价结论，判定被评价对象实现目标的程度，对比后鞭策后进、鼓励先进，激励不同层次群体 |
| 探索功能 | 评价过程对资料的收集与分析，是对事物深层次再认识的过程，可逐步发现事物发展规律，形成科学认识 |

## 一、评价尺度
评价时需先引进和确定评价尺度（标度），再通过评价尺度对评价对象进行判定并确定其价值。常见的评价尺度类型有：
- **绝对尺度**：其测定数值本身有意义。
- **间隔尺度**：只强调尺度点的意义。
- **顺序尺度**：用数字或反映顺序的字符表示，要求数字和对象相对应。
- **名义尺度**：仅仅是为了区别而分类的需要。

## 二、评价的基本原则
评价体现了主体在实施评价活动过程中的基本思想、精神和理念，或说侧重点和中心。在不同指导原则下进行评价，会得到不同的评价结果。评价的原则一般包括以下几方面：
- **客观性原则**：在评价过程中，要以评价标准为依据和基础，基于评价对象的基本特征，实事求是、客观、公正地评价，而不掺杂个人的主观臆断。评价必须客观地以存在的大量事实为依据，一切评价都产生于个别的情况，而不是开头，开头必先有观察，而后再去整理对个别事实特性的“知觉”，再确定评价中其系统的大小、不明确。
- **系统性原则**：系统地把评价对象从系统的整体，在相联系性、动态性和有序性等特点出发，遵循全面的观点（防止片面）、相互联系的观点（以防止孤立）、发展的观点（以防止静止僵化）来进行评价，使评价更准确、更系统、更深化。
- **分类与个性化原则**：评价是一个复杂的过程，包含多种因素、多重关系，因此，评价过程中应该根据评价对象的不同属性和特点确定相应的评价标准、评价指标和方法，进行分类评价，而不是将不同特点和不同特性的对象硬放在同一标准下评价，采用相同的评价方式和价值。由于被评价对象具有不同的个性，因而具有不可比性，所以，在评价过程中，必须对评价对象进行科学分类和“质”的分析，这样才能最大限度地使评价具有可比性，从而保证评价结果的科学性和可靠性。
- **科学性原则**：科学地规范评价的方法、标准、程序和结果的科学性和可靠性，评价过程和结果的可重复性。可重复性是指按照相同的评价过程、相同的评价方法连续地评价同样对象时，可得到同一结果的可能性大，那么评价结果的科学性、可靠性就高。由于各种因素的影响，评价结果往往是一种概率事件。但是，评价过程越公正，方法越科学，结果的趋同性和精确性、评价的价值也就越高。
- **实用性与可行性原则**：因此，评价在一定条件下进行的，这既要满足评价结果，又因为在实际时不太大地增加实施费用，因此，应尽量将有效的评价原理简单化，更为实践活动以及实用以多种可操作性，加之新的方法和评价指标十分复杂，不可能面面俱到，所以必须的因素相关、关系发展更为解释的就是合理的价值和力量的评价指标。
