# 系统动力学 

系统动力学（System Dynamic，SD）是一种定性分析与定量分析相结合的仿真技术，它以反馈控制理论为基础，以计算机仿真技术为手段，主要用于研究复杂系统的结构、功能与动态行为之间的关系，使决策者尝试各种情境下采取不同措施并观察模拟结果。

 # **1 方法介绍**

## 1.1 系统动力学的发展

系统动力学的研究在国外开展得比较早，发展也较为成熟。无论是理论研究方面还是实践应用方面，均有大量学者及科研机构对其进行研究，并取得相应的成果。国内对系统动力学的研究主要集中在系统建模和实践应用方面。虽然起步较晚，但经过多年的发展，系统动力学在我国区域和城市规划、组织研究、产业研究、科技管理等方面的研究与应用已取得了一定的成绩。

## 1.2 系统动力学的基本元素

构成系统动力学模型的基本元素包含流（Flow）与元素（Element）。流可分为实体流（Material Flow）和信息流（Information Flow）；元素包括状态变量（Level）、速率（Rate）和辅助变量（Auxiliary）。

## 1.3 系统动力学的组件

系统动力学建模有三个重要组件：因果反馈图、流图和方程式。

（1）因果反馈图：描述变量之间的因果关系；

（2）流图：用符号表达模型的复杂概念；

（3）方程式：系统动力学模型的结构主要由微分方程式所组成，每一个连接状态变量和速率的方程式即是一个微分方程式。

## 1.4 系统动力学的建模流程

系统动力学建模程序分为以下几个步骤：

1. 了解问题、界定问题、确认目标；
2. 确定系统边界，找到系统中的反馈回路，找到其中的状态变量和速率，确定结构，绘制系统的因果反馈图；
3. 建立系统动力学模型；
4. 测试模型、确认模型是否可以再现真实系统的行为；
5. 使用模型进行策略的选择；
6. 执行策略。
